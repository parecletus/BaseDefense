# BaseDefense
<img src="./BaseDefense.gif" width="400" align="left" style="margin-right:20px;">

## Overview
- This is a game where the player must protect their base.  
- Enemies drop upgrades when defeated.  
- Built as a small project to practice C# and game programming.

## Features
- Player movement using arrow keys.
- AABB collision detection and grid-based closest enemy search.
- Bullet count, speed, and other stats can be upgraded with Power-ups.
- Controls are explained in the in-game menu.
- New Power-ups can be implemented easily using delegates.
- Game.cs manages all systems, improving scalability.

## Tech Stack
- **Language:** C#  
- **Framework:** .NET 10.0  
- **Libraries:** Raylib-cs  
- **Development Tool:** Visual Studio
